การ find document ที่มีการเก็บข้อมูลในรูปแบบ array

คำสั่ง db.(ชื่อ collection).find("ชื่อ field":"[value]")
เช่น db.(ชื่อ collection).find({"social":["facebook",line,ig]}) //หากต้องการทราบ value มากกว่า 1 ค่าให้ใช้ , ขั้น
ผลลัพธ์จะแสดงข้อมูล document ที่ต้องการทราบโดยอ้างอิงถึง value ใน array

**ข้อควรระวัง ในการค้นข้อมูลที่เป็นรูปแบบของ array จะต้องออกแบบหรือกรอกข้อมูลตามลำดับตัวเลข index ที่จะค้นหาให้ถูกต้องจึงจะแสดงข้อมูล
